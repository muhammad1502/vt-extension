## ğŸ“¦ How to Install (Developer Mode)

Since this is a developer tool, you can install it without the Chrome Web Store:

1. **Download:** Click the green **Code** button above and select **Download ZIP** (or download from [Releases](https://github.com/muhammad1502/vt-extension/releases)).
2. **Unzip:** Extract the downloaded folder to a safe location.
3. **Open Extensions Menu:**
   - **Chrome/Brave:** Go to `chrome://extensions`
   - **Edge:** Go to `edge://extensions`
4. **Enable Developer Mode:** Toggle the switch in the top-right corner.
5. **Load:** Click **Load unpacked** and select the unzipped folder.
